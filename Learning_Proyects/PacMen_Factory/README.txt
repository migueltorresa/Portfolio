#PacMen Factory Exercise

This repository has the necesary files to create a website that uploads pacmen on command of a button and starts to bounce them around the edges of the page.
To run this project accordingly you need to click on the buttons that appear on the html file when opened on a explorer. 
This project runs on HTML, CSS and JavaScript.
A possible add-on for this project could be a button to restart or pause the game.
